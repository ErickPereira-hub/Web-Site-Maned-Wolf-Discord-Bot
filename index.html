Maned Wolf Discord Bot is a tool designed to track and analyze activity across any Discord server. It integrates Discord’s native database with its own, recording and processing operations such as member deletions, message updates, and much more. The bot automatically analyzes server data, generates detailed reports on key elements, and even provides predictive insights, forecasting future trends in membership growth and message activity.

🚀 Objectives of the bot

  • 📌 Includes a command that generates a report of member joins, leaves, and the total server count over a user‑defined period. In addition, the report highlights server growth by comparing recent activity with the server’s overall lifecycle.
  • 📌 Offers a command that estimates upcoming member counts through polynomial regression and reports the margin of error in percentage terms
  • 📌 Provides commands that calculate the probability of gaining a user‑defined range of new members on the following day (ex.: 10 to 15 members). Similarly, it can evaluate the probability of reaching a specified range of message volume (ex.: 400 to 500 messages) for the next day. These features are implemented using the Poisson distribution
  • 📌 Includes a command that identifies the most active members across the entire server, as well as a command that does the same thing within a specific channel selected by the user.
  • 📌 Provides a command that identifies the most active channels in the server, ranked by message volume.
  • 📌 Includes a command that displays the volume of channels based on NSFW and non-NSFW.
  • 📌 The commands allow the user to see graphical figures that display the data.
